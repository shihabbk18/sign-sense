# Sign Sense - ASL Detector

Real-time **American Sign Language (ASL)** alphabet detector using **Convolutional Neural Network (CNN)** and OpenCV.

## Features
- Real-time sign language detection from webcam
- Trained on ASL Alphabet dataset (29 classes)
- Clean and simple GUI
- High training accuracy

## Tech Stack
- **TensorFlow / Keras** (CNN Model)
- **OpenCV** (Real-time video processing)
- **MediaPipe** (Optional - for future improvements)
- Python
